⚙️ EINRICHTUNG – SCHRITT FÜR SCHRITT

Folge diesen Schritten um dein Dashboard auf allen Geräten zu synchronisieren.

SCHRITT 1 · GITHUB ACCOUNT ERSTELLEN

1. Gehe zu github.com und klicke auf "Sign up"
2. Erstelle einen kostenlosen Account
3. Verifiziere deine E-Mail-Adresse
4. Erstelle ein neues Repository: Klicke auf "+" → "New repository"
5. Name: phd-life-os, stelle es auf Public, klicke "Create repository"
6. Lade diese HTML-Datei hoch: "uploading an existing file" → Datei rein ziehen → "Commit changes"
7. Gehe zu Settings → Pages → Source: "main" → Save
8. Deine URL: deinname.github.io/phd-life-os

SCHRITT 2 · FIREBASE EINRICHTEN

1. Gehe zu firebase.google.com und melde dich mit deinem Google-Account an
2. Klicke auf "Projekt hinzufügen" → Name: phd-life-os
3. Google Analytics: kann deaktiviert werden → "Projekt erstellen"
4. Klicke auf "Web" (das </> Symbol) → App-Name: "PhD Life OS" → "App registrieren"
5. Du siehst jetzt einen firebaseConfig Block mit deinen Daten – kopiere ihn
6. Im linken Menü: "Erstellen" → "Realtime Database" → "Datenbank erstellen"
7. Wähle den Standort Europe-west1 → "Weiter"
8. Wähle "Im Testmodus starten" → "Aktivieren"

SCHRITT 3 · CONFIG EINFÜGEN

1. Öffne diese HTML-Datei in einem Texteditor (z.B. TextEdit auf Mac, Notepad auf Windows)
2. Suche nach: apiKey: "DEIN_API_KEY"
3. Ersetze den gesamten firebaseConfig Block mit deinen kopierten Daten aus Firebase
4. Speichere die Datei
5. Lade die aktualisierte Datei erneut auf GitHub hoch (gleicher Prozess wie Schritt 1)

SCHRITT 3b · SICHERHEITSREGELN SETZEN 🔒

1. In Firebase Console → Realtime Database → Tab "Regeln"
2. Ersetze alles mit diesem Text:

{
  "rules": {
    "phdOS": {
      "users": {
        "$uid": {
          ".read": "$uid === auth.uid",
          ".write": "$uid === auth.uid"
        }
      }
    }
  }
}

3. Klicke "Veröffentlichen"
4. Außerdem: Authentication → Sign-in method → Google aktivieren ✅
→ Jetzt kann nur noch du mit deinem Google-Account deine Daten lesen!

// So sieht dein Config aus (Beispiel):
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "phd-life-os.firebaseapp.com",
  databaseURL: "https://phd-life-os-default-rtdb...",
  projectId: "phd-life-os",
  ...
};
SCHRITT 4 · ALS APP INSTALLIEREN

📱 iPhone / iPad:
1. Öffne deine GitHub Pages URL in Safari
2. Tippe auf das Teilen-Symbol (Quadrat mit Pfeil nach oben)
3. Scrolle runter → "Zum Home-Bildschirm"
4. Name: "PhD Life OS" → "Hinzufügen"
5. Die App erscheint auf deinem Homescreen wie eine echte App! ✅

💻 Mac / Windows:
1. Öffne die URL in Chrome
2. Klicke auf das Install-Symbol in der Adressleiste (Computer mit Pfeil)
3. "Installieren" → erscheint im Dock / Startmenü
